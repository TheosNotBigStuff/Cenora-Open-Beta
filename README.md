# Welcome to the Cenora Open Beta
With this repositry you can try out Cenora versions before they fully release, please don't use it as a main desktop enviorment.

# How to download a beta
1. Go to our releases tab
2. Download an beta
3. Unzip the file
4. Open the 'cenora.exe' file and enjoy.
